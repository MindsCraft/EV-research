## 📌 Verification Checklist Before Any Purchase

| Step | Action |
|:---|:---|
| ✅ Certificate check | TÜV: certipedia.com · CE: EU NANDO database · OCPP: openchargealliance.org |
| ✅ Spare parts test | Ask: *"Can we buy individual modules and PCBs separately?"* — If no, walk away |
| ✅ Surge protection | Confirm built-in Class I + II SPD (essential for Bangladesh monsoon lightning) |
| ✅ Sample order | Order 2–5 units of each model before any bulk commitment |
| ✅ OCPP test | Connect sample to your CPMS — verify remote start/stop, billing, OTA update |
| ✅ Factory audit | For orders above $25,000: commission SGS, TÜV, or Bureau Veritas (~$500–1,500) |
